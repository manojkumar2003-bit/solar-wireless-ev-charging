# Solar Wireless Electric Vehicle Charging System

Mini Project | Bangalore Institute of Technology | Dept. of EE(VDT) | 2024-25

## Overview
A solar-powered wireless EV charging system using inductive power 
transfer (WPT) and IoT-based monitoring via ESP8266 and Blynk platform.

## Components Used
- Solar Panels (10W, 5V)
- ESP8266 (NodeMCU) microcontroller
- Wireless transmitter & receiver coils
- Blynk IoT platform
- SANCA batteries (4V, 1.6Ah)
- Voltage sensor, LCD display

## Key Results
- Charging time: ~1 hour 11 minutes
- Solar energy generated: 40Wh/day
- Wireless transfer efficiency: 85%
